CORE FEATURES:

1. SYNTHETIC DATA GENERATION
   ─────────────────────────────────────────────────────────────────────
   • Generates 500+ realistic FIR document images
   • 10 IPC section classes (50 samples each)
   • Automatic augmentation:
     - Gaussian noise (2-8% variance)
     - Blur (kernel sizes 3, 5, 7)
     - Rotation (-5 to +5 degrees)
     - Perspective distortion
     - Scan artifacts (shadows, dust)

2. IPCV PREPROCESSING PIPELINE (6 Stages)
   ─────────────────────────────────────────────────────────────────────
   Stage 1: Noise Reduction
           Non-Local Means Denoising (h=10, preserve details)
   
   Stage 2: Contrast Enhancement
           CLAHE (Contrast Limited Adaptive Histogram Equalization)
   
   Stage 3: Perspective Correction
           Edge-based document boundary detection & rotation correction
   
   Stage 4: Binarization
           Adaptive Gaussian Thresholding (blockSize=11)
   
   Stage 5: Morphological Operations
           Closing → Opening → Dilation (clean up & connect)
   
   Stage 6: Edge Detection
           Canny (50-150) + Dilation (strengthen edges)

3. FEATURE EXTRACTION (65+ Features)
   ─────────────────────────────────────────────────────────────────────
   • HOG Features:                  ~180 features
   • LBP Features:                  ~59 features
   • Edge Features (Sobel):         7 features
   • Contour Features:              13 features
   • Statistical Features:          11 features
   • Morphological Features:        8 features
   • Text Region Features:          6 features
   • Frequency Domain Features:     4 features
   ────────────────────────────────────────────
   TOTAL:                           260+ after scaling

4. MACHINE LEARNING MODEL
   ─────────────────────────────────────────────────────────────────────
   Algorithm:           Random Forest Classifier
   Number of Trees:     200
   Max Depth:           20
   Min Samples Split:   5
   Min Samples Leaf:    2
   Number of Classes:   10 (IPC sections)
   Number of Features:  260+
   Training Samples:    400
   Test Samples:        100
   Expected Accuracy:   75-85%

5. INFERENCE & PREDICTION
